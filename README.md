# Laboration 4 - TypeScript
I denna laboration har jag som student skapat en webbapplikation där användaren kan se Webbutvecklingsprogrammets kurset, sortera dem, samt söka efter specifika kurser.

För att genomföra detta har ramverket Angular använts, samt en HttpClient. För styling har SCSS använts.
